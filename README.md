# 🎮 ArcadeWorld — веб-платформа мини-игр

## 📌 Цель проекта

Разработать веб-приложение, в котором пользователь может:
- играть в классические мини-игры прямо в браузере
- сохранять результаты
- просматривать таблицу лидеров

**Основная цель:**  
продемонстрировать навыки разработки интерактивных веб-приложений на чистом:
- HTML
- CSS
- JavaScript (без сторонних библиотек)

---

## 🧩 Описание проекта

**ArcadeWorld** — это агрегатор игрового контента.

Платформа отвечает за:
- регистрацию игрока
- навигацию между играми
- хранение рекордов
- отображение таблицы лидеров

Игры реализуются как **независимые модули**, изолированные в отдельных директориях.

Проект можно расширять как Open Source платформу:  
для добавления новой игры достаточно создать папку по стандартам API ArcadeWorld.

---

## 🔌 Требования к игровым модулям

Каждая игра должна:

- быть изолированной (не изменять глобальные переменные)
- экспортировать стандартные функции
- содержать файл метаданных:
  - название
  - автор
  - описание
  - путь к иконке

## 🎯 Список игр

### 🧱 Arkanoid
- управление платформой
- отбивание мяча
- разрушение блоков
- начисление очков

### 🐍 Game
- управление змейкой
- сбор еды
- увеличение длины
- завершение при столкновении

### ❌⭕ Tic-Tac-Toe
- классические крестики-нолики
- игра против компьютера
- проверка победителя

### ⚡ Reaction Game
- клик по объекту как можно быстрее
- измерение времени реакции

### 🦖 Dino Runner
- аналог оффлайн-игры в браузерах

> ⚠️ Список игр может изменяться

## 🖥️ Требования к интерфейсу

### 🏠 Главный экран
- название проекта
- поле ввода ника
- список игр
- кнопка перехода к таблице лидеров
- краткое описание проекта

---

### 🎮 Страница игры
- игровое поле
- текущий счёт
- кнопка старта / рестарта
- отображение состояния игры
- сообщение о завершении

---

### 📊 Экран результатов
- отображение результата
- сохранение результата
- кнопка новой игры
- переход к таблице лидеров

---

### 🏆 Таблица лидеров
- список игроков
- ник игрока
- лучший результат
- сортировка по очкам
- отдельные рейтинги для каждой игры

## ⚙️ Дополнительные требования
